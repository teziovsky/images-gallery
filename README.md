## [Demo](https://teziovsky.github.io/images-gallery/)

## Run locally

Clone the project and go to the project directory

```bash
git clone https://github.com/teziovsky/images-gallery.git && cd images-gallery
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run serve
```
